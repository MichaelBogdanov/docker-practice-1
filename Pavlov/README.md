# Scoreboard API

**Студент:** Павлов  
**Задание:** топ по очкам для разных игр

## Технологии
- Python 3.11 + Flask
- PostgreSQL 15
- Docker, docker-compose

## Запуск проекта
1. Убедитесь, что Docker Desktop установлен и запущен.
2. Перейдите в папку Pavlov.
3. Выполните команду:
   docker-compose up --build
