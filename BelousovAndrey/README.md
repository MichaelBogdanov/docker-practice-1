В ходе прохождения практической работы была реализована программа "Рецепты"
Программа разрабатывалась на языке Python с использованием библиотеки Flask для связи с API
Были реализованы все указанные решения и протестированы с использованием Docker

Реализованны API запросы "POST /recipes", "GET /recipes", "GET /recipes/random"
И тестирование с docker compose up --build
