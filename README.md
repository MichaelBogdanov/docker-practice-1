# Image Metadata API

HTTP API для хранения мета-информации об изображениях (URL, теги, размеры).

## Функциональность

- POST /images - добавление нового изображения
- GET /images - получение списка всех изображений (с возможностью фильтрации по тегу)
- GET /images/{id} - получение детальной информации об изображении

## Технологии

- ASP.NET Core 8.0 Web API
- In-memory хранилище (данные хранятся в памяти приложения)
- Docker контейнеризация
- Docker Compose для оркестрации
