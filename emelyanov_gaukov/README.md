# Сервис сокращения ссылок

## Описание
Простой HTTP API для создания коротких ссылок, редиректа и просмотра статистики.
Реализован на Python без внешних зависимостей.
Данные хранятся в JSON-файле, сохраняющемся между перезапусками благодаря Docker Volume.

## Запуск
1. Убедитесь, что Docker Desktop установлен и запущен.
2. Откройте терминал в корневой папке проекта.
3. Выполните команду для сборки и запуска:
```
docker compose up --build
```

После запуска сервис будет доступен по адресу: http://localhost:5000

Для остановки нажмите Ctrl+C или выполните:
```
docker compose down
```

## API Endpoints
1. Создать короткую ссылку
Метод: POST /shorten
Принимает JSON с длинным URL, возвращает уникальный короткий код.

Пример запроса:
```
curl -X POST http://localhost:5000/shorten -H "Content-Type: application/json" -d "{\"url\":\"https://www.youtube.com/watch?v=dQw4w9WgXcQ\"}"
```

Пример ответа:
```
{
  "code": "a1b2"
}
```

2. Редирект по коду
Метод: GET /r/{code}
Выполняет HTTP 302 Redirect на оригинальную ссылку и увеличивает счётчик просмотров.

Пример запроса:
```
curl -v http://localhost:5000/r/a1b2
```
Результат:
Перенаправление браузера или клиента на исходный URL.

3. Статистика перехода
Метод: GET /stats/{code}
Возвращает информацию о ссылке: оригинальный URL, дату создания и количество просмотров.

Пример запроса:
```
curl http://localhost:5000/stats/a1b2
```

Пример ответа:
```
{
  "code": "a1b2",
  "original_url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
  "created_at": "2026-05-07T12:00:00.000000",
  "views": 5
}
```
