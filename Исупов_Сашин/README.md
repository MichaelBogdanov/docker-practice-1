# Jokes API

## Выполнили
Исупов, Сашин

## Реализованные возможности

- Получение списка всех шуток
- Фильтрация шуток по категориям
- Получение случайной шутки
- Добавление новых шуток
- Голосование за шутки (up/down)

## Техническая реализация

- **Язык программирования:** C# (.NET 9.0)
- **Веб-фреймворк:** ASP.NET Core
- **Контейнеризация:** Docker, Docker Compose
- **Хранение данных:** в памяти приложения

## Запуск проекта

```bash
docker compose up --build
