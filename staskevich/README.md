# Вариант 9 — Таблица рекордов

## Участники
Стаскевич

## Описание
Реализован HTTP API для хранения рекордов игроков по разным играм.

Поддерживаются endpoint'ы:
- `POST /scores` — добавить результат игрока
- `GET /scores/top?game=digdug&limit=10` — получить топ результатов по игре
- `GET /scores/player/{name}` — получить все результаты выбранного игрока

Данные хранятся в памяти приложения в виде списка записей.
Каждой новой записи автоматически присваивается уникальный `id`.
